Termes

  • - Unit : chaque mot délimités par des espaces, sous la forme 1x0042
  • - Opcode : type d'unit, détermine l'opération a effectuer
  • - Befx : la partie de l'unit avant x (Before x)
  • - Aftx : la partie après x (After x)
  • - LKA : l'interface graphique de l’interpréteur

Units

  • - 0x : Opcode (2 Aftx)
  • - 1x : Direct (4)
  • - 2x : Reférence (4)
  • - 9x : Misc (4)

Opcodes

  • Mémoire
    • x00 : STOR  (2ope)
    • x01 : MOV   (2ope)
    • x00 : COP   (2ope)
  • Arithmétique
    • x10 : ADDT  (2ope)
    • x11 : SOUS  (2ope)
    • x12 : MULT  (2ope)
    • x13 : DIVI  (2ope)
  • Comparaison
    • x20 : EQU   (2ope)
    • x21 : DIFF  (2ope)
    • x22 : SUP   (2ope)
    • x23 : INF   (2ope)
    • x23 : SUPE  (2ope)
    • x23 : INFE  (2ope)
  • Bool
    • x30 : AND   (2ope)
    • x31 : OR    (2ope)
    • x32 : XOR   (2ope)
    • x33 : NOT   (1ope)
  • Branchement
    • x40 : IF    (2ope)
    • x41 : IFN   (2ope)
    • x42 : GOTOR (1ope)
    • x43 : GOTOG (1ope)
  • Misc
    • x50 : NOPE  (2ope)
    • x51 : PRINT (2ope)

Utilisation Mémoire

  • - 0 : Code Pointer
  • - 1 : Résulat d'opération Arithmétique
  • - 2 : Résultat de comparaison
  • - 3 : Résultat d'opération de logique Booléenne

LKA Shortcuts

  • F1 : Commente la Sélection
  • F5 : Recharge la page
  • F9 : Run the code
  • Alt+C : Change Log
  • Alt+A : About
